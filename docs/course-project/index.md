# Course Project

Build an LLM agent that designs and tunes a controller for an aircraft attitude-control plant.

- **[Aircraft Attitude Control Agent](llm-agent.md)** — the project itself: system description, parameters, design specs, and deliverables.
- **[Beginner Template](beginner-template.md)** — *optional reference* for students new to Python: install instructions, project file layout, and minimal Streamlit / Gradio skeletons with TODO markers. Alternative tools and layouts are welcome.
- **[Demo](demo.md)** — a complete worked example end-to-end on a textbook second-order plant: full backend + frontend code, captured agent run, converged step-response plot. Use it as a template for your own sub-option.
- **[Q&A](qa.md)** — common questions about the project, updated as they come in.
